========================================================================
NEON ARENA - MOD LOADER & MOD PACK MANAGER
========================================================================

Welcome to the Neon Arena Modding directory. Here you can place your customized mods:
1. .al files: Arena Loader configuration files. These are parsed in real-time
   by our server to customize game physics, gravity, and bot properties.
2. .cs files: Unity C# scripting mods.
3. .exe files: Standalone wrapper executables that boot the game with mod pre-injection.

USE THE IN-GAME MODDING AND CONTROLLER STUDIO:
You can create mods inside the web UI, compile them into .al scripts or standalone Windows .exe binaries, load them directly onto your active multiplayer lobby, and share them with other players.

========================================================================
CONTROLLER SUPPORT:
- Customize controller button mappings.
- Test active gamepads live inside the config panel.
- Adjust stick deadzones and look sensitivities.
