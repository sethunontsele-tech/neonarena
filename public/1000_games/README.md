# 1000 Games Folder

This folder is designated for your offline HTML games. Please drag and drop your game `.zip` packages here!

To fulfill your request, there are **no games preloaded inside this filesystem folder** to keep it completely pristine. You can sideload and decompress up to 1,000 games dynamically using the browser-based Cabinet emulator, which handles loading, local indexing, and offline playback automatically.

## Storage Cap
- Pre-allocated cap of **60.00 GB** for custom titles.
